https://public.cyber.mil/stigs/gpo/

Download the latest GPO package from the DoD Cyber Exchange and extract to this folder. The extracted structure should look similar to below:

.\GPO
├───ADMX Templates
│   ├───Adobe
 (...)
│   └───OneDrive
│       └───en-us
├───DoD Adobe Acrobat Pro DC Classic V1R3
 (...)
├───DoD Windows Server 2019 MS and DC V2R2
└───Support Files
    └───Local Policies
        ├───DoD Windows 10 v2r2
        └───DoD Windows Server 2019 MS v2r2
